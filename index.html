First Column: Short

This div should only stick to the top as it is too short to stick to bottom and top.

It uses the following code:

$('#short-column').stickable({
margin: 20,
bottomId: 'footer'
});

Introduction

Stickable.js is a fixed position responsive sidebar that solves a lot of the problems we've run into with all the other plugins we've used in the past.

  • It's responsive out of the box
  • It has simple methods to reset the fixed positioning to account for changes in the DOM.
  • It has start and stop setting for fixed positioning (see right hand sidebar)
  • It has an intutive scrolling behavior that follows the directions your are scrolling

We'll be posting more soon but check out the source of this page to see how it works

marginBottom: 100,
bottomId: 'footer'
